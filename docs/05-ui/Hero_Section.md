# 🔷 LAYER 2 — VISUAL DIRECTION UPGRADE

---

## 2.1 Background System (UPGRADED)

### Old:

- Static dark + tiny grid

### New:

> **Layered Depth Background (3 layers)**

### Structure:

```
Layer 1: Base dark (#0b0f19)
Layer 2: Soft radial glow (very subtle)
Layer 3: Micro grid pattern
```

### Rules:

- Add **very soft blue radial glow** behind content
  - opacity: 6–10%
  - position: slightly off-center (not perfectly centered)

- Grid stays, but:
  - blur slightly
  - reduce sharpness

### Effect:

👉 Feels **alive + premium**
👉 NOT flashy, but no longer “flat”

---

## 2.2 Add “Focus Glow” Behind Content

A soft light behind headline area:

- Color: primary blue
- Blur: high
- Opacity: very low

**Purpose:**

- Draws eye to center automatically
- Makes hero feel designed, not placed

---

## 2.3 Noise Layer (Optional but Recommended)

Very subtle grain texture (1–2% opacity)

👉 Adds **real-world depth**
👉 Removes “perfect digital flatness”

This is the biggest upgrade.

### What:

Very subtle animated elements in background:

- small dots / nodes
- connected lines (optional)
- slow movement

### Behavior:

- opacity: low

### Meaning:

- structure

## 5.2 Motion Rules

| ----------- | --------- |
